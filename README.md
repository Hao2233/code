### 1. 初始化git仓库
```
git init
```
### 2. git提交到缓存区
```
git add README.md
git add --all
```
### 3. git提交到工作区
```
git commit -m "first commit"
git commit -m -a
```
### 4. git提交到远程仓库GitHub
```
git push
```
### 5. 查看git提交历史
```
git log
```
### 6. 查看git状态
```
git status
```
### 7. 查看git修改信息
```
git diff
```
